Tutorial: MK Web X als Standard-Browser einstellen

  1. Öffne den Browser und klicke oben rechts auf Extras → MK Web X.
  2. Wähle Als Standard-Browser registrieren.
  3. Wechsle in Windows zu Einstellungen → Apps → Standard-Apps.
  4. Gib in der Suchleiste MK Web ein und wähle es als Standard.
